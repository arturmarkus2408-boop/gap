# GAP — AI-тренажёр переговоров

> **GAP** (узб. *гап*) — разговор. Личный AI-собеседник для отточки навыков переговоров.

Создан для владельца бухгалтерской и консалтинговой компании в Ташкенте. Адаптирован под узбекскую деловую практику: ООО/ИП, ГНК, ЕНП, типичные клиентские возражения.

---

## Что умеет приложение

- **4 типа сценариев:** продажа услуг, удержание клиента, переговоры с партнёром, переговоры с банком/налоговой
- **4 уровня сложности:** от лёгкого до жёсткого собеседника
- **Голосовой ввод** (микрофон) — диктовка ответа вместо набора текста
- **Режим «за рулём»** — ИИ озвучивает ответы вслух и сам включает микрофон
- **Подсказки** — три варианта твоей следующей фразы, если затрудняешься
- **Разбор сессии** — структурированный анализ: что сильно, что слабо, что улучшить

## Команды в чате

| Кнопка | Что делает |
|---|---|
| 💡 | Подсказка — три варианта моей следующей фразы |
| 🎤 | Голосовой ввод — один раз нажал, сказал, текст появился |
| 🔊 (в шапке) | Режим «за рулём»: озвучка + автомикрофон |
| 📋 Разбор | Структурированный анализ всей сессии |

**Текстовые команды** (можно написать в чат во время сессии):
- `сложнее` — собеседник станет жёстче
- `мягче` — собеседник снизит напор

## Где взять API-ключ

Бесплатный ключ Gemini: **aistudio.google.com/app/apikey** → «Create API key»

Лимит: ~1500 запросов в день. Одна сессия = 20-40 запросов = 30-50 сессий в день.

Ключ хранится только на твоём телефоне.

## Как обновить приложение

1. Скачать новые версии файлов
2. На GitHub: **Add file → Upload files** → перетащить → **Commit changes**
3. На телефоне: **Настройки → Приложения → Chrome → Хранилище → Очистить данные**
4. Открыть ссылку заново → переустановить иконку

## Если что-то сломалось

**Не работает ИИ, ошибка про модель** — Gemini сменил название модели. Открыть `index.html` на GitHub → найти строчку с `gemini-2.5-flash` → заменить на актуальную из списка моделей aistudio.google.com.

**Приложение показывает старую версию** — очистить данные Chrome в Настройках Android.

**Chrome поверх иконки** — удалить иконку → очистить данные Chrome → открыть ссылку → 30 секунд походить по интерфейсу → меню Chrome → «Установить приложение».

**Микрофон не работает** — Настройки Android → Приложения → Chrome → Разрешения → Микрофон → Разрешить.

**Озвучка молчит** — проверить громкость медиа на телефоне (не звонка, а медиа).
